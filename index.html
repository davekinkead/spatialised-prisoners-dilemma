Spatialised Prisoner's Dilemma

This is an agent based simulation of an spatialised evolutionary prisoner's dilemma. A prisoner's dilemma is a game where players can either cooperate or defect with each other.

Each action has a payoff based on the other player's action and there are 8 possible deterministic strategies given an initial move, a response to cooperation, and a response to defection.

  • Always Defect - red
  • Suspicious Opposite - purple
  • Suspicious Tit-for-Tat - indigo
  • Defect, then Cooperate - lime
  • Cooperate, then Defect - orange
  • Friendly Opposite - pink
  • Friendly Tit-for-Tat - blue
  • Always Cooperate - green

For every tick of the simulation, agents plays 10 rounds of a prisoner's dilemma with others in their neighbourhood. At the end of 10 rounds, agents adapt the best performing strategy from their neighbourhood.

The simulation shows that while defection is initially successful, friendly tit-for-tat will dominate the simulation when ever a cooperative cluster develops.

Click on the simulation to start / stop it. Detailed explanation and code is available on github.
